# PROJECT BRIEF — Invisibrain AI

## One-Line Summary
> "A stealth desktop AI assistant that listens, sees your screen,
>  and whispers real-time answers during interviews, meetings,
>  and coding exams — completely invisible to screen-sharing tools."

## Problem Statement
- Online interviews create high-pressure situations where candidates
  freeze despite knowing the material
- Meeting participants miss key points and struggle with real-time responses
- Coding exam candidates need instant algorithmic guidance
- Current AI tools (ChatGPT, browser extensions) are VISIBLE
  to screen-sharing and proctoring software
- No existing tool combines voice + screen + AI + stealth in one package

## Target Users
| Persona              | Description                                    |
|----------------------|------------------------------------------------|
| Job Candidates       | Software engineers in technical interviews      |
| Meeting Attendees    | Professionals in high-stakes business meetings  |
| Students             | Taking online proctored coding exams            |
| Sales Professionals  | Needing real-time objection handling             |

## Core Value Proposition
1. **INVISIBLE** — Physics-level stealth, bypasses ALL screen capture
2. **ZERO SETUP** — No Python, no Whisper, no dependencies
3. **REAL-TIME** — Sub-500ms AI responses via Gemini cascade
4. **ALL-IN-ONE** — Voice + Screen + Code + Resume context
5. **PANIC BUTTON** — Emergency vanish in milliseconds

## Success Metrics
- [ ] App invisible on Zoom/Teams/Meet screen share (100% stealth)
- [ ] Voice-to-answer pipeline < 3 seconds end-to-end
- [ ] Screen capture to code solution < 5 seconds
- [ ] Zero crashes during 2-hour interview sessions
- [ ] Executable size < 100MB
- [ ] Launch time < 2 seconds
- [ ] 1,500 daily API requests without rate limiting
